**Lex Launcher – combined feature & fix prompt for Replit:**

1. **Default launcher** – Convert Lex into a full launcher app that takes over the home screen UI and can be set as the default launcher (implement required Android `HOME`/`LAUNCHER` intent filters and launcher activity).

2. **Accurate real-time data** – Lex must pull and report real device data: current date/time from the system clock, weather from a live weather API, and device temperature/battery stats from system sensors/APIs. No guessing, caching stale values, or false confirmations — if data isn't available, say so.

3. **Persistent sign-in & sessions** – Once a user signs up or signs in on a device, never show the sign-up/sign-in screen again on that device (persist auth session securely). Sessions should also not expire or reset during normal app-to-app transitions (e.g. opening the media/image picker and returning to the lock screen) — only end on explicit logout, device restart, or token expiry.

4. **Launcher-level device access** – Request and properly handle permissions (installed apps, notifications, app usage stats, contacts/calls if needed) so Lex can read and act on device data as a launcher.

5. **Screen lock recovery via Clerk** – Screen lock (PIN/password/pattern, optional Face ID) recovery must go through Clerk authentication, not Google account recovery. Verify identity via Clerk, then allow resetting the lock.

6. **Core launcher features** – Implement standard launcher functionality: home screen with app grid/widgets, app drawer, notification badges, search, wallpaper support, gesture navigation, and settings to manage default launcher status.

7. **Custom & built-in widgets** – Allow users to describe a widget in natural language and have Lex generate/configure it. Provide a built-in library of 30 widgets covering varied sizes (small, medium, large) and functions — examples: clock, weather, calendar, battery, notes, to-do list, music player, calculator, photo frame, search bar, app shortcuts folder, contacts, fitness/steps, screen time, alarm, timer, news feed, stock ticker, recent apps, storage usage, signal/wifi status, flashlight toggle, sticky note, countdown, quote of the day, habit tracker, mail preview, messages preview, currency converter, unit converter.

8. **Branding** – Replace all v0 logo references in `manifest.json` (icons, favicons, app icons) with the Lex orb logo across all required sizes.
